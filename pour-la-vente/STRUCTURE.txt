🎯 WORKFLOW CLIENT (CE QU'ILS FONT)
═══════════════════════════════════════════════════════════════════════

1. 📥 RECEVOIR
   └─→ Télécharger le ZIP ou recevoir la clé USB

2. 📖 LIRE
   └─→ Ouvrir "LISEZ-MOI-EN-PREMIER.txt"

3. 📱 PRÉPARER
   └─→ Réinitialiser le téléphone (sans compte Google)
   └─→ Activer le débogage USB

4. 🔌 CONNECTER
   └─→ Brancher le téléphone en USB à l'ordinateur

5. 💻 INSTALLER
   └─→ Windows : Double-clic sur "INSTALLER-ZITELOU-WINDOWS.bat"
   └─→ Mac : Double-clic sur "installer-zitelou-mac.sh"
   └─→ Linux : Double-clic sur "installer-zitelou-linux.sh"

6. ✅ CONFIGURER
   └─→ Créer compte parent + code PIN
   └─→ Ajouter contacts autorisés
   └─→ Sélectionner applications

⏱️ DURÉE TOTALE : ~10 MINUTES

🛠️ CE QUE FONT LES SCRIPTS AUTOMATIQUEMENT
═══════════════════════════════════════════════════════════════════════

Les 3 scripts (Windows/Mac/Linux) effectuent automatiquement :

1. ✅ Détection d'ADB (Android Debug Bridge)
2. ✅ Installation d'ADB si nécessaire
3. ✅ Vérification de la connexion du téléphone
4. ✅ Installation de l'application Zitelou
5. ✅ Activation du mode Device Owner (contrôle parental)
6. ✅ Attribution des permissions nécessaires
7. ✅ Configuration du launcher par défaut
8. ✅ Lancement de l'application

→ LE CLIENT N'A QU'À SUIVRE LES INSTRUCTIONS À L'ÉCRAN !

PROBLÈMES FRÉQUENTS :

1. "Téléphone non détecté"
   → Vérifier câble USB, débogage USB, changer de port

2. "Device Owner échoue"
   → Téléphone pas vierge, réinitialiser sans compte

3. "Script bloqué par sécurité"
   → Windows/Mac : Clic droit > Exécuter quand même
