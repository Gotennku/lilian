📱 VOUS AVEZ DANS CE DOSSIER :

   ✅ zitelou.apk                    → L'application à installer
   ✅ INSTALLER-ZITELOU-WINDOWS.bat  → Pour Windows (double-clic)
   ✅ installer-zitelou-mac.sh       → Pour Mac (double-clic)
   ✅ installer-zitelou-linux.sh     → Pour Linux (double-clic)
   ✅ MODE-EMPLOI-COMPLET.pdf        → Guide illustré avec photos


🚀 INSTALLATION RAPIDE (5 MINUTES) :

   1. PRÉPAREZ LE TÉLÉPHONE (important !)
      → Réinitialisez-le complètement (paramètres d'usine)
      → Ne mettez AUCUN compte Google
      → Activez le "Débogage USB" (voir le guide)

   2. CONNECTEZ LE TÉLÉPHONE
      → Branchez le téléphone à votre ordinateur avec un câble USB

   3. LANCEZ LE SCRIPT
      → Windows : Double-cliquez sur "INSTALLER-ZITELOU-WINDOWS.bat"
      → Mac : Double-cliquez sur "installer-zitelou-mac.sh"
      → Linux : Double-cliquez sur "installer-zitelou-linux.sh"

   4. SUIVEZ LES INSTRUCTIONS
      → Le script s'occupe de tout automatiquement !


⚠️ TRÈS IMPORTANT :

   ► Le téléphone DOIT être réinitialisé (vierge)
   ► SANS compte Google configuré
   ► Sinon l'installation échouera !
