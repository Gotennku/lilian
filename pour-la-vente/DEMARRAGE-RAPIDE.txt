# 🎯 DÉMARRAGE RAPIDE - 3 ÉTAPES SEULEMENT

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│              INSTALLATION ZITELOU EN 3 ÉTAPES                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## ÉTAPE 1 : PRÉPARER LE TÉLÉPHONE (5 min)

```
📱 SUR LE TÉLÉPHONE À SÉCURISER :

┌─────────────────────────────────────────────────────────────────┐
│  1. RÉINITIALISER                                               │
│     Paramètres > Système > Réinitialisation > Tout effacer     │
│                                                                 │
│  2. AU REDÉMARRAGE                                              │
│     ✅ Langue : Français                                        │
│     ✅ Wi-Fi : Connecter                                        │
│     ⛔ Compte Google : SKIP (ne pas ajouter)                   │
│     ⛔ Compte Samsung : SKIP (ne pas ajouter)                  │
│                                                                 │
│  3. ACTIVER LE DÉBOGAGE USB                                     │
│     a) Paramètres > À propos du téléphone                       │
│     b) Taper 7 FOIS sur "Numéro de build"                       │
│     c) Message : "Vous êtes développeur"                        │
│     d) Paramètres > Options développeurs                        │
│     e) Activer "Débogage USB"                                   │
└─────────────────────────────────────────────────────────────────┘
```

---

## ÉTAPE 2 : CONNECTER (30 sec)

```
🔌 BRANCHER LE TÉLÉPHONE À VOTRE ORDINATEUR :

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   [Téléphone] ──────USB────── [Ordinateur]                     │
│                                                                 │
│   Sur le téléphone, popup :                                     │
│   "Autoriser le débogage USB ?"                                 │
│                                                                 │
│   ✅ Cocher : "Toujours autoriser"                             │
│   ✅ Cliquer : "OK"                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## ÉTAPE 3 : LANCER LE SCRIPT (2 min)

```
💻 SUR VOTRE ORDINATEUR :

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  WINDOWS :                                                      │
│  📁 Double-clic sur :  INSTALLER-ZITELOU-WINDOWS.bat           │
│                                                                 │
│  MAC :                                                          │
│  📁 Double-clic sur :  installer-zitelou-mac.sh                │
│                                                                 │
│  LINUX :                                                        │
│  📁 Double-clic sur :  installer-zitelou-linux.sh              │
│                                                                 │
│  ──────────────────────────────────────────────────────────    │
│                                                                 │
│  LE SCRIPT S'OCCUPE DE TOUT :                                   │
│  • Installe ADB automatiquement                                 │
│  • Installe Zitelou sur le téléphone                            │
│  • Active le contrôle parental                                  │
│  • Configure tout pour la sécurité                              │
│                                                                 │
│  VOUS N'AVEZ QU'À SUIVRE LES INSTRUCTIONS !                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## ✅ C'EST TERMINÉ !

